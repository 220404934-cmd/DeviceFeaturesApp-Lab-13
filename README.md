## Açıklama
Bu proje, Mobil Programlama dersi kapsamında verilen React Native Lab
dokümanına uygun olarak geliştirilmiştir.

Laboratuvarda istenen tüm adımlar uygulanmış, gerekli kütüphaneler
yüklenmiş ve örnek kodlar birebir kullanılmıştır.
Ancak uygulama çalıştırılırken Android / Expo Go ortamında
teknik bir hata oluşmuş ve uygulama çalışmamıştır.

## Yüklenen Kütüphaneler
- expo-image-picker
- expo-camera
- expo-location
- expo-notifications
- expo-haptics
- @react-navigation/native
- @react-navigation/native-stack
